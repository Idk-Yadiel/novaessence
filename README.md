# Nova Essence

Pagina web estatica para promocionar productos artesanales Nova Essence, incluyendo combo especial RD$800, vela aromatica de canela, gel hidratante y exfoliante de cafe.

## Estructura

- `index.html`: contenido principal de la pagina.
- `css/styles.css`: estilos, diseno responsive y animaciones.
- `app.js`: menu, slider, formulario de WhatsApp, quiz, FAQ y contador fijo.
- `*.png`: imagenes usadas por la pagina.

## Publicacion

Puedes publicar esta carpeta en cualquier hosting estatico como Netlify, Vercel, GitHub Pages o un servidor tradicional.

El archivo principal que debe abrir el navegador es:

```text
index.html
```
